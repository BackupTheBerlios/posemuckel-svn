Servereinstellungen

Die Namen von Posemuckelservern kannst Du auf der Posemuckel-Website erfahren.

Trage den Namen eines Servers unter Host Server ein, und den dazugehörigen Port unter Port. Standardmäßig wird der Port 8081 benutzt, aber er kann von Server zu Server unterschiedlich sein. Der Port wird normalerweise zusammen mit dem Servernamen angegeben. Sollte nichts angegeben sein, dann versuche es mit 8081.

Zum Einloggen auf den ausgewählten Server mußt Du wieder zum Tab Einloggen wechseln und Einloggen drücken.

Die Anwendung merkt sich alle eingegebenen Server und Ports, so dass Du bequem den Server wechseln kannst, indem Du ihn Dir aus der Drop-Down-Liste aussuchst. Um den Server zu wechseln, mußt Du Dich natürlich erst ausloggen, den Servereintrag ändern und dann wieder einloggen.

Gegebenenfalls mußt Du Dich erneut registrieren, da jeder Server eine eigene Benutzerdatenbank unterhält. Hast Du z.B. bisher immer posemuckel.no-ip.org benutzt und wählst nun posemuckel.dnsalias.org als Server, kennt dieser Dich noch nicht.
