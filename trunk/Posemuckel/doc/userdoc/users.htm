Benutzer

Ein registrierter Benutzer kann folgende Aktionen ausführen:

  • an Projekten teilnehmen
  • selber ein Projekt starten
  • mit anderen Benutzern chatten

Die LED vor dem Benutzernamen leuchtet grün, wenn der Benutzer gerade online ist. Erscheint gleichzeitig noch ein P in der grünen LED, so bedeutet dies, dass er gerade in einem Projekt aktiv ist.
Ist sie grau, ist der Benutzer offline, und erscheint in einer grauen LED ein Fragezeichen, dann weiß der Server nicht so recht Bescheid ;-).
