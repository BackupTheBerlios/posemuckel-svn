Benutzer finden

In der Projektübersicht kann man im Menü über Benutzer -> Benutzersuche den Dialog zur Benutzersuche aufrufen. Dort kann man alle registrierten Benutzer durch bestimmte Kriterien filtern. Zur Verfügung stehen die Felder Vorname, Nachname, Username, Sprache und Geschlecht. Einträge in mehrerer dieser Felder werden mit einem logischen UND verknüpft: Eine Suche nach einem Vornamen "Jens" und dem Geschlecht "weiblich" wird wahrscheinlich kein Ergebnis haben!

Besonderer Erläuterung bedarf das Feld Freitextsuche : Hier können alle Benutzerdaten (außer Wohnort und Emailadresse) gleichzeitig abgefragt werden. Eine Freitextsuche nach "Klaus" findet sowohl Benutzer mit eben jenem Vornamen, Usernamen oder Nachnamen. Die Freitextsuche schließt übrigens auch das Feld Kommentar eines Benutzers ein.

Unterstützung von Wildcards

In allen Feldern können sogenannte Wildcards benutzt werden, als Wildcards gelten folgende Zeichen:

* steht für eine beliebige Anzahl von Zeichen
? steht für genau ein Zeichen

Beispiele:
Eine Suche nach: im Feld: findet:
?arina Vorname Carina, Karina, Marina, ...
M*ler Nachname Müller, Möller, Moeller, Maler, Meinke-Stadler, ...
?arina* Username Karina26, carina1977, Marinade, ...
Ich * ?ein ?acho*ein! Freitextsuche Benutzer mit Kommentar:
Ich bin kein Machoschwein!,
Ich will Dein Nacho sein!, ...

In der Freitextsuche kann eine Suche mit Wildcards unter Umständen ein reichhaltiges Ergebnis liefern!
